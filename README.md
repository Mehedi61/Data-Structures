  ### How to run
  Open your Linux terminal and execute the commands below.  
  ```
  git clone https://github.com/Mehedi61/Data-Structures.git
  ```
  ```
  cd Data-Structures/
  ```
  for python ...
  ```
  python3 filename.py
  ```
  for chsarp ...
  ```
  mcs filename.cs
  ```  
  ```  
  mono filename.exe
  ```
